20. JDBC

20.1 Introduction of JDBC :-

JDBC is the acronym of Java Detabase Connectivity

JDBC is an API which consists of a set of java classes, interfaces and Exception.

With the help of JDBC API we can request a connectio with database the send the query statement using SQL and receive the results for processing.

JDBC supports two packages.

  1. java.sql.*
  2. javax.sql.*

JDBC supports four drivers.

  1. JDBC-ODBC bridge driver
  2. Native-API driver
  3. Net-Protocol driver
  4. Native-Protocol driver

20.2 JDBC API :-

JDBC API is a connectivity between the java programming language and databases.

JDBC API makes three things :-

  1. Establish a connection with a database or access any tabular data sources.
  2. Send SQL statements.
  3. Process the results.

JDBC API :-

  1. java.sql.connection :- Interface used to establish a connection.
  2. java.sql.Driver :- Interface used to locate the driver for a particular database management system.
  3. java.sql.statement :- Interface used to send static SQL statements to the database server and obtain results.
  4. java.sql.preparedStatement :- Interface used to send pre compiled SQL statements to the database server and obtain results.
  5. java.sql.ResultSet :- Interface used to process the results returned from executing on SQL statement.
  6. java.lang.DriverManager :- Class used to manage a set of JDBC Drivers.
  7. java.sql.Exception :- Exception that provides information about a database error.

20.3 JDBC Drivers :-

JDBC Driver is a software component that enable the java application to interact with a database.

20.4 Steps to connect the program to the database :-

  1. import the package
  2. Load the driver
  3. Establish the connection
  4. Create the statement
  5. Execute the statement
  6. Process result
  7. Close the connection
Example :- 
import java.sql.*;
class Demo {
    public static void main(String[] args) {
        Class.forName("com.mysql.jdbc.Driver");
        Connection con = DriverManager.getConnection("jdbc:mysql://localhost:3306/db_name","root","password");
        Statement st = con.CreateStatement();
        ResultSet rs = st.executeQuery("Select * from emp");
        while(rs.next()){
            String name = rs.getString("emp_name");
            int sal = rs.getInt("sal");
            System.out.println(name + " " + sal);
        }
        st.close();
        con.close();
    }
}
