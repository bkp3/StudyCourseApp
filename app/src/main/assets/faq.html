Frequently Asked Questions

1. What is Java?

Java is the high-level, object-oriented, robust, secure programming language, platform-independent, high performance, Multithreaded, and portable programming language. It was developed by James Gosling in June 1991. It can also be known as the platform as it provides its own JRE and API.

2. Explain about Java Virtual Machine?

JVM is a program that interprets the intermediate Java byte code and generates the desired output. It is because of byte code and JVM that programs written in Java are highly portable.

  5. Mutli threaded
  6. Robust
  7. Secure

4. Differentiate between JVM, JRE, and JDK

JDK provides the environment to develop and execute the java program.JDK contains two parts.

One part contains javac, which convert source code into byte code. Second part is JRE which contains utilities like java which help in running the code

JRE provides the libraries, The JVM and other components to run only java byte code.

JVM is a software. It provides runtime environment in which java bytecode can be executed.

5. Does Java Use Pointers?

No. Java doesn't use pointers. It has a tough security. Instead of pointers, references are used in Java as they are safer and more secure when compared to a pointer.
