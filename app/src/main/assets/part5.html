5. Java String

5.1 String :-

Java implements sring as a object. String are immutable objects in java.

When we create an object of string class using new operators, it creates string object in heap memory area.

When we create an object of string class using string literal, it creates string object in string constant pool area.

In heap, we can create two or more objects of string class having identical character.

In SCP, duplicate string objects are not allowed.

Example :- 
class Demo {
    public static void main(String[] args) {
        String s1 = "java";
        Char ch = {'p', 'r', 'o'};
        String s2 = new String(ch);
        String s3 = new String("App");
        System.out.println(s1);
        System.out.println(s2);
        System.out.println(s3);
    }
}

Output:- 
        Java
        pro
        App

5.2 String methods :-

java.lang.string provides so many methods.

  1. concat()
  2. length()
  3. toUpperCase()
  4. toLowerCase()
  5. indexOf()
  6. charAt()
  7. equals()
  8. substring()

4.3 concat() :-

concat() method is used to add the strings.

Example :- 
class Demo {
    public static void main(String[] args) {
        String str1 = "Hello";
        String Str2 = "World";
        System.out.println(str1 + " " + str2);
    }
}

Output:- Hello World

4.4 length() :-

The length of a string can be found with the length() method.

Example :- 
class Demo {
    public static void main(String[] args) {
        String str = "Hello";
        System.out.println(str.length());
    }
}

Output:- 5

4.5 toUpperCase() :-

It is used to convert the string to upper case.

Example :- 
class Demo {
    public static void main(String[] args) {
        String str = "Hello";
        System.out.println(str.toUpperCase());
    }
}

Output:- HELLO

4.6 toLowerCase() :-

It is used to convert the string to lower case.

Example :- 
class Demo {
    public static void main(String[] args) {
        String str = "Hello";
        System.out.println(str.toLowerCase());
    }
}

Output:- hello

4.7 indexOf() :-

The indexOf() method returns the position of the first occurrence of a specified text in a string.

Example :- 
class Demo {
    public static void main(String[] args) {
        String txt = "Please locate where 'locate' occurs!";
        System.out.println(txt.indexOf("locate"));
    }
}

Output:- 7

4.8 charAt() :-

charAt() method returns a char value at the given index number.

Example :- 
class Demo {
    public static void main(String[] args) {
        String str="world";
        char ch=str.charAt(2);
        System.out.println(ch);
    }
}

Output:- r

4.9 equals() :-

equals() method compares the two given strings based on the content of the string.

Example :- 
class Demo {
    public static void main(String[] args) {
        String str1 = "Hello";
        String Str2 = "Hello";
        System.out.println(str1.equals(str2));
    }
}

Output:- true

4.10 substring() :-

substring() method returns a part of the string.

Example :- 
class Demo {
    public static void main(String[] args) {
        String str="javatutorial";
        System.out.println(str.substring(3,6));
    }
}

Output:- atut
