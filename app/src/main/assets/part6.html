6. Oops Concept

6.1 Introduction :-

Oops stands for object oriented programming language.

It is the way to do programming with the help of class and object.

oops provides the ability to simulate real world event.

By using oops concepts we can provide the solution of real world problem.

The main aim of object oriented programming is to implement real world entities.

Basic concepts of oops

  • Object
  • Class
  • Inheritance
  • Polymorphism
  • Abstraction
  • Encapsulation

6.2 Object :-

Any entity that have state and beheviour is known as object.

Example :-

Phone is a object.
It has state such as price, color, weight, dimensions etc.
It has behaviour such as calling, messaging etc

6.3 Class :-

Collection of objects is called class.

The properties of class are called attributes.

The behaviour of class is called method.

Example :- 
class Employee {
    int id;
    String name;
    void showDetails(){
        System.out.println(id);
        System.out.println(name);
    }
    public static void main(String[] args) {
        Employee e = new Employee();
        e.id = 123;
        e.name = "Hello";
        e.showDetails();
    }
}

Output:- 
        123
        Hello

6.4 Inheritance :-

When one class acquire all the properties and behaviours of a parent class then it is known as inheritance.

6.5 Polymorphism :-

If one task is performed in different ways than it is known as polymorphism.

To achieve polymorphism we use method overloading and method overriding.

6.6 Encapsulation :-

Binding code and data together into a single unit is kknown as encapsulation.

Example :-

Capsule, It is bind with different medicine.

Class is example of encapsulation

6.7 Abstraction :-

Hiding internal details and only showing its functionality is known as abstraction.

To achieve abstraction we use abtract class and interface.

Example :-

Phone call, we dont know the internal processing.
