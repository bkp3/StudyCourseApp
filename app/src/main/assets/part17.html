17. final keyword

17.1 Introduction :-

In java programming final can be :-

  1. variable
  2. method
  3. class

17.2 final variable :-

In java to make constant we use final keyword.

If you make any variable as final, we can not change the value of that variable.

Example :- 
class Demo {
    final int x = 20;
    void fun(){
        x = 30;
    }
}

Output:- compile time error

17.3 final method :-

If we make any method as final, we can not override it.

Example :- 
class Parent {
    final void show() {
        System.out.println("Parent show");
    }
}

class Child extends Parent{
     void show() {
        System.out.println("Child show");
    }
}

Output:- compile time error

17.4 final class :-

If we make any class as final we can not extend it.

Example :- 
final class Parent {
    void show() {
        System.out.println("Parent show");
    }
}

class Child extends Parent{
     void show() {
        System.out.println("Child show");
    }
}

Output:- compile time error
