2. Java Fundamentals

2.1 Naming Conventions in Java :-

  • Naming convention of class and interface :- The first letter should be capital and if several words linked together to form the name. the first letter of the inner words should be upper case.

    Example :- class Student{}, class Account{}, class PrintWriter{}, interface Runnable{} etc.

  • Naming convention of methods :- The first letter should be lower case and then normal camelCase rule should be used.

    Example :- getBalance(), doCalculation(), getCustomerName() etc.

  • Naming convention of Variables :- Naming conventions of variables name is same as methods. the first letter of the inner words should be upper case.

    Example :- age, empName, employee_net_sal etc.

  • Naming convention of Constants :- Yhe constants are always capital (Upper case letter) with underscore (_) characters as seperators.

    Example :- MIN_PRIORITY, PI, MIN_HEIGHT etc.

  • Naming convention of keywords :- All keywords should be written by using all small letters as follows:

    Example :- public, void, static etc.

2.3 Variables in Java :-

A variable represents the memory location in which value can be stored.

The following are the variables in java:

  1. Instance variable:

    Instance variables are the member of an object of a class, which can only accessible through the object, like (object name.(variable name)) (obj.a = 5)

  2. Class variable/Static variable:

    Class variable are the member of an object of a class, which is declared as static and which can be access before creation of object and also after creation of object.
    Example :- static int a;.

  3. Local variable:

    The variable that are declared with in a method or constructor is called local variable and can't be accessed by other method.
    Example :- void calculate(int a, int b){ }

2.5 Operators in Java :-

The following are the operators supported in Java.

i Arithmetic Operators :-

  • + :- Additive operator (also used for string concatenation)
  • - :- Substraction operator
  • * :- Multiplication operator
  • / :- Division operator
  • % :- Remainder operator

    • Example :- 
    class Demo {
    public static void main(String[] args) {
        int x = 10;
        int y = 20;
        int result = x + y;
        System.out.println("x + y = " + result);
        result = x - y;
        System.out.println("x - y = " + result);
        result = x * y;
        System.out.println("x * y = " + result);
        result = y / x;
        System.out.println("y / x = " + result);
        result = x % 3;
        System.out.println("x % y = " + result);
    }
}
    
Output:- 
        x + y = 30
        x - y = -10
        x * y = 200
        y / x = 2
        x % y = 1

ii Assignment Operators :-

  • = :- Simple assigment operator
  • += :- It adds right operand to the left operand and asssign the result to left operand.
  • -= :- It substracts right operand to the left operand and asssign the result to left operand.
  • *= :- It multiplies right operand to the left operand and asssign the result to left operand.
  • /= :- It divides left operand to the right operand and asssign the result to left operand.
  • %= :- It takes modulus using two operands and asssign the result to left operand.

iii Unary Operators :-

  • + :- unary plus operator: indicates positive value
  • - :- unary minus operator: negates an expression
  • ++ :- Increment operator: increment a value by 1
  • -- :- Decrement operator: decrements a value by 1
  • ! :- Logical complement operator: inverts the value of a boolean

iv Relational Operators :-

  • == :- Equal to
  • != :- Not equal to
  • > :- Greater than
  • >= :- Greater than or equal to
  • < :- Less than
  • <= :- Less than or equal to

v Conditional Operators :-

  • && :- Conditional - AND
  • || :- Conditional - OR
  • ?: :- Ternary (shorthand for if-else statement)

vi Type Comparison Operators :-

  • instanceof :- Compare an object to a specified type. (This operator is used only for object reference type (class type or interface type))
    • Example :- 
    class Demo {
    public static void main(String[] args) {
        String name = "James";
        boolean result = name instanceof String;
        System.out.println(result);
    }
}
    
Output:- 
        true

vii Bitwise and Bit shift Operators :-

  • ~ :- Unary bitwise complement
  • << :- Signed left shift
  • >> :- Signed right shift
  • >>> :- Unsigned right shift
  • & :- Bitwise AND
  • ^ :- Bitwise exclusive OR
  • | :- Bitwise inclusive OR

    • Example :- 
    class Demo {
    public static void main(String[] args) {
        int x = 5;
        int y = 6;
        System.out.println("~x =  " + (~x));
        System.out.println("x & y = " + (x & y));
        System.out.println("x | y = " + (x | y));
        System.out.println("x ^ y = " + (x ^ y));
        System.out.println("x >> 1 = " + (x >> 1));
        System.out.println("x << 1 = " + (x << 1));
        System.out.println("x >>> 1 = " + (x >>> 1));
    }
}
    
Output:- 
        ~x = -6
        x & y = 4
        x | y = 7
        y ^ x = 3
        x >> 1 = 2
        x << 1 = 10
        x >>> 1 = 2

viii The new Operators :-

  • new :- The new operator is used for create the object dynamically for the class.

2.6 Input/output in java :-

In java we can take the data from the keyboard by using the following three ways.

  1. Using Scanner Class
  2. Using InputStreamReader class
  3. Through Command Line argument

Example i:- Using Scanner Class of java.util package 
import java.util.*;
class Demo {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("Enter First Number:");
        int a = sc.nextInt();
        System.out.println("Enter Second Number:");
        int b = sc.nextInt();
        int sum = a + b;
        System.out.println("Addition of " + a + " and " + b + " = " + sum);
    }
}

Input:- 
        Enter First Number: 5
        Enter Second Number: 10
Output:- 
        Addition of 5 and 10 = 15
Example ii:- Using BufferedReader class of java.io package 
import java.io.*;
class Demo {
    public static void main(String[] args) {
        InputStreamReader in = new InputStreamReader(System.in);
        BufferedReader br = new BufferedReader(in);
        System.out.print("Enter your name: ");
        String str = br.readLine();
        System.out.println("I know, Your name is " + str);
        System.out.print("Enter a number: ");
        int x = Integer.parseInt(br.readLine());
        System.out.println("The number is " + x);
    }
}

Input:- 
        Enter your name: Java
        Enter a number: 10
Output:- 
        I know, Your name is Java
        The number is 10
