14. Encapsulation

14.1 Introduction :-

Binding of variables and methods into a single unit is known as encapsulation.

Encapsulation and data hiding are used synonymously everywhere but it should not be misunderstand that encapsulation is all about data hiding only.

To achieve encapsulation in java :-

  • We declare the variables of class as private.
  • We provides public setter and getter methods.
Example :-

    private String name;
