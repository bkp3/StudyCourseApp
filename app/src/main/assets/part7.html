7. Constructors

7.1 Introduction :-

Java provides a special method called constructor.

A constructor intializes instance members.

Its name is same as the class name.

Constructor can be overload.

It may have any accessibility public, protected, private.

The main purpose of constructor is to initialize the variables.

Types of Constructor:

  • Default Constructor
  • Parameterized Constructor

7.2 Default constructor :-

Constructor with zero paremeter called default constructor.

Example :- 
class Demo {
    Demo(){
        System.out.println("Default Constructor");
    }
    public static void main(String[] args) {
        Demo obj = new Demo();
    }
}

Output:- Default Constructor

7.3 Parameterized Constructor :-

Constructor with one or more parameter called parameterized constructor.

Example :- 
class Demo {
    Demo(int a){
        System.out.println("Parameterized Constructor");
    }
    public static void main(String[] args) {
        Demo obj = new Demo(5);
    }
}

Output:- Parameterized Constructor
