11. Access Modifiers

11.1 Introduction :-

The access modifiers in java specifies the accessibility or scope of a variable, method or class.

We can change access level of variable, method, constructor and class by applying modifiers on it.

There are four types of access modifiers.

  • Public
  • Protected
  • Default
  • Private

11.2 Public :-

The access level of a public modifier is everywhere.

It can be accessed from within the class, outside the class, within the package and outside the package.

11.3 Protected :-

The access level of protected modifiers is within the package and outside package through child class.

If you do not make the child class. It can not be accessed from outside the package.

11.4 Default :-

The access level of a defalut modifiers is only within the package.

It can not be accessed from outside the package.

If you do not specify any access level. It will be the default.

11.5 Private :-

The access level of a private modifier is only within the class. It can not be accessed from outside the class.
