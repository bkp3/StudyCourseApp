12. Inheritance

12.1 Introduction :-

When one class acquires all properties and behaviour of its parent class then it is known as inheritance.

The most important feature of inheritance is code re-usability.

The mechanism of creating a new class from an old class is called inheritance.

old class :- Parent, Base, Super.

new class :- Child, Derived, Sub.

Example :- 
class Parent {
    void show() {
        System.out.println("Parent");
    }
}

class Child extends Parent{
     void display(){
        System.out.println("Child");
     }
     public static void main(String[] args){
        Child obj = new Child();
        obj.show();
        obj.display();
     }
}

Output:- 
        Parent
        Child

12.2 Types of Inheritance :-

There are several types of inheritance.

  1. Single inheritance
  2. Multilevel inheritance
  3. Multiple inheritance

12.3 Single inheritance :-

One base class and one derived class.

Example :- 
class Parent {
    void show() {
        System.out.println("Parent");
    }
}

class Child extends Parent{
     void display(){
        System.out.println("Child");
     }
     public static void main(String[] args){
        Child obj = new Child();
        obj.show();
        obj.display();
     }
}

Output:- 
        Parent
        Child

12.4 Multilevel inheritance :-

Derived class further inherited by another class

Example :- 
class Parent {
    void show() {
        System.out.println("Parent");
    }
}

class Child extends Parent {
    void show() {
        System.out.println("Child");
    }
}

class GrandChild extends Child{
     void display(){
        System.out.println("Grand Child");
     }
     public static void main(String[] args){
        Child obj = new Child();
        obj.show();
        obj.display();
        obj.fun();
     }
}

Output:- 
        Parent
        Child
        Grand Child

12.5 Multiple inheritance :-

Java does not support multiple inheritance at class level.
